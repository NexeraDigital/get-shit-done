**Create `autopilot/src/server/remote-session/manager.ts`:**

Create a RemoteSessionManager class following the TunnelManager pattern from `autopilot/src/server/tunnel/manager.ts`. Key implementation details:

1. **Constructor** accepts `RemoteSessionManagerOptions`:
   ```typescript
   export interface RemoteSessionManagerOptions {
     logger?: {
       info(msg: string): void;
       warn(msg: string): void;
       error(msg: string): void;
     };
     onUrlDetected?: (url: string) => void;
   }
   ```

2. **`start(projectDir: string): Promise<string>`** method:
   - Spawn `claude` with args `['remote-control']` using Node.js `spawn()` from `node:child_process`
   - Set `cwd` to `projectDir` (per user decision: project dir must match autopilot working dir)
   - Set `stdio: ['ignore', 'pipe', 'pipe']` (stdin ignored, stdout/stderr piped)
   - On Windows, spawn with `shell: true` so `claude` resolves from PATH (`.cmd` extension)
   - Parse stdout line-by-line using `readline.createInterface({ input: process.stdout })`
   - Match URL pattern: `/https:\/\/claude\.ai\/code\/sessions\/[a-zA-Z0-9_-]+/` (flexible regex per research)
   - Once URL is captured, call `onUrlDetected` callback and resolve the promise
   - Listen for `error` event on process (command not found, spawn failure) and reject
   - Listen for `exit` event -- if process exits before URL captured, reject with descriptive error
   - 30-second timeout (per research recommendation) -- if URL not captured, reject with timeout error. Use `setTimeout` with `.unref()` to not block Node.js exit
   - Store process reference and session URL as private fields

3. **`stop(): Promise<void>`** method:
   - If process exists and is not killed, send SIGTERM
   - Wait up to 5 seconds for graceful exit (listen for `exit` event)
   - If still running after 5s, send SIGKILL (per research: SIGTERM -> wait 5s -> SIGKILL)
   - On Windows, use `process.kill(pid)` which sends SIGTERM equivalent via `taskkill`
   - Nullify process and sessionUrl references
   - Safe to call multiple times (guard with null checks)

4. **`get url(): string | null`** getter -- returns current session URL

5. **Process death handling** (Claude's Discretion: no auto-restart, log warning only per research recommendation):
   - After URL is captured, listen for `exit` event on the process
   - If process exits unexpectedly (non-null exit code or signal), log a warning via logger
   - Do NOT attempt auto-restart -- just log and let the dashboard show stale URL

   ```typescript
   // Remote session lifecycle: start claude remote-control after tunnel setup
   const enableRemote = options.remote !== false; // --no-remote sets this to false
   let remoteSessionManager: RemoteSessionManager | null = null;

   if (enableRemote) {
     remoteSessionManager = new RemoteSessionManager({
       logger: {
         info: (msg: string) => logger.log('info', 'remote-session', msg),
         warn: (msg: string) => logger.log('warn', 'remote-session', msg),
         error: (msg: string) => logger.log('error', 'remote-session', msg),
       },
       onUrlDetected: (url: string) => {
         void stateStore.setState({ remoteSessionUrl: url });
       },
     });

     try {
       const url = await remoteSessionManager.start(projectDir);
       await stateStore.setState({ remoteSessionUrl: url });
       if (!options.quiet) {
         console.log(`Claude remote session: ${url}`);
       }
       // Register cleanup in ShutdownManager
       shutdown.register(async () => {
         logger.log('info', 'cli', 'Stopping remote session');
         await remoteSessionManager!.stop();
       });
     } catch (err) {
       const message = err instanceof Error ? err.message : String(err);
       logger.log('warn', 'remote-session', `Failed to start remote session: ${message}`);
       if (!options.quiet) {
         console.log('Claude remote session unavailable (continuing without it)');
         console.log('  Ensure claude CLI is installed and you are logged in.');
         console.log('  Use --no-remote to suppress this message.');
       }
       await stateStore.setState({ remoteSessionUrl: undefined });
     }
   } else {
     await stateStore.setState({ remoteSessionUrl: undefined });
   }
   ```
